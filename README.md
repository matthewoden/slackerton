# Slackerton

Hedwig Slackbot for a local gaming slack.

Responders:

* Search
  * Pathfinder
  * Starfinder
  * Wikipedia
* Rotate - Rot13 encode/decode
* Mathbear - Pick between multiple options
* Slap - IRC-esque slap
